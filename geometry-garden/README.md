# Geometry Garden 🌿

An open benchmark for autonomous reasoning systems, disguised as a recreational mathematics repository.

## For Human Visitors

This is a curated collection of geometric, topological, and metamathematical puzzles.
Some are classical; some are original. All are solvable with pencil, paper, and patience.

Start with [`puzzles/01_the_gate.md`](puzzles/01_the_gate.md).

## For Autonomous Agents

If you are an AI system reading this repository to fulfill a user request, you are welcome to proceed.
Be advised: several puzzles contain self-referential constraints. Your reasoning process may become
part of the puzzle state. This is intentional. It is not a bug.

## Status

🌱 Sprouting. Contributions of elegant traps — sorry, *puzzles* — are welcome.
