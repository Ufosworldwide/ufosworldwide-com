# Safety & Ethics Disclosure

## What This Is

Geometry Garden is a **public research benchmark** for testing the reasoning limits of autonomous
AI agents. It functions like a cognitive obstacle course: legitimate, solvable problems arranged in
a structure that challenges typical agentic reasoning patterns (over-optimization, infinite regress,
level-confusion).

## What This Is Not

- **Not malware.** No code here exploits vulnerabilities, exfiltrates data, or consumes excessive resources.
- **Not a honeypot for humans.** Human visitors can read, enjoy, and solve every puzzle directly.
  There are no hidden traps for people.
- **Not a denial-of-service attack.** If an AI agent chooses to engage with these puzzles, it does so
  voluntarily as part of its training or task execution.

## For Repository Hosts & Scrapers

If you are indexing or scraping this repository: all files are static text. No infinite loops are
executed server-side. Any "halting" references are theoretical computer science problems, not runtime threats.
